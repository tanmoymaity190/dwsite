There is many cases with configuration & security purposes in Kali Linux. First of all you have to install gpg key and repository.
There is two gpg key named SHA256SUMS and SHA256SUMS.gpg. These key are available on KALI official website. After download it you have install. Open QTerminal and follow the steps...

$ gpg --verify SHA256SUMS.gpg SHA256SUMS
$ wget -q -O - https://kali.org/archive.key.asc | gpg --import

Now create repository to get system & security update. Open QTerminal and enter the following command...

$ touch /etc/apt/sources.list
$ chmod 644 /etc/apt/sources.list
$ echo "deb https://http.kali.org/kali kali-rolling main contrib non-free" > /etc/apt/sources.list
$ sudo apt-get update
$ sudo apt-get upgrade -y
$ sudo apt-get full-upgrade -y

Congrats! You are done GREAT!

To restart all services with new version make your system restart.

$ reboot

Anonmouse Mode

For safty & privecy of user you have to do operation anomously from your system......
Fire up kali linux and open the QTerminal. Then execute the command

$ git clone https://github.com/SusmithKrishnan/torghost.git
$ cd torghost
$ pip3 install -r requirements.txt
$ chmod +x build.sh && ./build.sh
$ chmod +x install.sh && ./install
$ sudo su
$ apt-get -f install

It is all set. Now you can start torghost or make traffic via tor.
Use command for for operation torhost.
torghost -s --start Start Torghost
torghost -r --switch Request new tor exit node
torghost -x --stop Stop Torghost
torghost -h --help Print this help and exit

Port Forwarding

Port Forwarding is a process to expose your local port to expose to the web. Via client software it communicate with the external server port. And data will be incoming & outgoing via it. Here I dicussed about only Ngrok client for port forwarding.

First of all download the client from Ngrok official site https://www.ngrok.com and Register with valid credentials and the make login. Then copy the ngrok authentication token.
After it open the QTerminal on kali linux.

$ sudo su
$ cd Downloads
$ unzip
$ mv ngrok /usr/bin
$ ngrok authtoken
$ ngrok http 80 (For HTTP port)
$ ngrok tcp 80 (For TCP port)
